## Getting Started

### Prerequisites
- PHP installed on your system (v7.4 or higher recommended).

### Running Locally
1. Clone or download this repository.
2. Open your terminal and navigate to the project root:
   ```bash
   cd /path/to/SeniorProjectProfile
   ```
3. Start the PHP built-in server:
   ```bash
   php -S localhost:8000
   ```
4. Access the site in your browser at `http://localhost:8000`.
